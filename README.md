Интернет-магазин косметики "adam beauty store"

Описание проекта
Современный интернет-магазин уходовой косметики с адаптивным дизайном, каталогом товаров, корзиной покупок и системой избранного.

Функционал
- Просмотр каталога товаров
- Фильтрация по категориям
- Добавление товаров в корзину
- Добавление товаров в избранное
- Оформление заказа
- Форма обратной связи с валидацией
- Адаптивный дизайн для всех устройств

Технологии
- HTML5 - семантическая разметка
- CSS3 - Flexbox, Grid, CSS переменные
- JavaScript (ES6+) - интерактивность
- LocalStorage - хранение данных
- Font Awesome - иконки
- Google Fonts - шрифты

Условия и циклы
- Фильтрация товаров по категориям (if/else)
- Обработка массива товаров (forEach, filter)
- Проверка наличия товара в корзине

Функции (12+ функций)
1. displayProducts() - отображение товаров
2. addToCart() - добавление в корзину
3. addToFavorites() - добавление в избранное
4. filterCatalog() - фильтрация товаров
5. updateCartCount() - обновление счетчика
6. showNotification() - уведомления
7. checkout() - оформление заказа
8. validateContactForm() - валидация формы
9. initMobileMenu() - мобильное меню
10. animateStatistics() - анимация статистики
11. getFavorites() - получение избранного
12. renderCartView() - отображение корзины

События
- Клики по кнопкам фильтров
- Отправка форм
- Изменение количества товаров
- Добавление в корзину/избранное

Как запустить
1. Склонировать репозиторий
2. Открыть index.html в браузере
3. Протестировать функционал:
   - Фильтрация товаров в каталоге
   - Добавление товаров в корзину
   - Добавление в избранное
   - Оформление заказа
   - Форма обратной связи
